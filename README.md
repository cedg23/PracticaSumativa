# Calculadora básica en PHP

## Descripción

Este proyecto es una práctica básica desarrollada en PHP que permite realizar operaciones matemáticas simples: suma, resta y multiplicación.
El programa se ejecuta directamente en PHP.

## Funcionalidades

* Suma de dos números
* Resta de dos números
* Multiplicación de dos números
* Muestra los resultados directamente en la salida del programa

## Tecnologías utilizadas

* PHP

## Cómo ejecutar el proyecto

1. Clonar o descargar este repositorio
2. Colocar el archivo `.php` en un servidor local (por ejemplo, XAMPP)
3. Abrir el navegador y acceder al archivo, por ejemplo:
http://localhost/nombre-del-archivo.php

## Ejemplo de uso

Si se definen los siguientes valores:

* Número 1: 10
* Número 2: 5

El programa mostrará:

* Suma: 15
* Resta: 5
* Multiplicación: 50
